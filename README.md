# Sales Dashboard (Power BI)

## Overview
This project presents an interactive sales performance dashboard built using Power BI.

## Objective
To analyze revenue, product performance, and regional sales trends to support business decision-making.

## Tools
- Power BI
- SQL
- CSV dataset

## Dashboard Features
- Total Sales KPI
- Total Orders KPI
- Total Quantity KPI
- Monthly Sales Trend
- Sales by Product
- Sales by Region
- Quantity by Product
- Interactive slicers for Region, Product, and Date

## Key Insights
- Identifies top-performing products
- Highlights regional sales differences
- Tracks sales trends over time
- Supports interactive analysis through slicers
